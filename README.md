# CENG140-THE2-Tester-Sec3

* CENG140 THE2 Tester for Section3
* Tester is written in Python2.7
* Since it includes some bash commands and gcc compiler, it is recommended to use Mac or Linux.

# How to use

1) Copy your the2.c (if this is not the case, rename it) to tester folder.
2) Run the tester by using following command: "python tester.py"
3) You can see which case you failed at and cases in input folder.
